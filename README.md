# Account Viewer

### Versions
Angular 13<br/>
Asp.net 7

## Run the angular app

```
npm install
```
```
ng serve
```
## Run the asp.net backend app

```
dotnet run
```

### Completed functionalities
Excel Upload: http://localhost:4200/account-upload <br/>
Account Data View: http://localhost:4200/account-view <br/>
