# Account Viewer

### Versions
Angular 16<br/>
Asp.net 7

## Run the angular app

```
npm install
```
```
ng serve
```
## Run the asp.net backend app

```
dotnet run
```


### How to use

There exist 2 users: <br/>
* Admin <br/>
 username: admin <br/>
 password: 123 <br/>
* User <br/>
  username: user <br/>
  password: 456 <br/>
  
As per the requirement only admin can access account uploads section while normal user can access only account view section.
